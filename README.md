# KDD2020-Expert-Linking
Pytorch implementation of the KDD2020 paper "Zero-Shot Expert Linking via Adversarial Contrastive Learning"

## Requirements
- Ubuntu 16.04
- Python 3.6
- Pytorch 1.1.0
- GPU 32G
